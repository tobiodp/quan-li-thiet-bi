# Giải thích tên dự án "ResourcesManagement"

## ResourcesManagement là gì?

**ResourcesManagement** là tên dự án của bạn - **Hệ thống quản lý tài nguyên (thiết bị)**.

## Cấu trúc tên:

- **Resources** = Tài nguyên (thiết bị, máy móc)
- **Management** = Quản lý
- **ResourcesManagement** = Quản lý tài nguyên

## Nơi sử dụng:

### 1. **Package name:**
```java
package com.example.ResourcesManagement.service;
package com.example.ResourcesManagement.entity;
package com.example.ResourcesManagement.repository;
```

### 2. **Application name:**
```properties
spring.application.name=ResourcesManagement
```

### 3. **Database name:**
```properties
spring.datasource.url=jdbc:mysql://.../ResourcesManagement
```

### 4. **Maven artifact:**
```xml
<artifactId>ResourcesManagement</artifactId>
```

### 5. **JAR file name:**
```
ResourcesManagement-0.0.1-SNAPSHOT.jar
```

## Cách trả lời khi bị hỏi:

**"ResourcesManagement là tên dự án của em, có nghĩa là Hệ thống quản lý tài nguyên. Dự án này dùng để quản lý thiết bị công nghệ như laptop, monitor, keyboard... trong công ty."**

## Tên tiếng Việt:

- **Tên dự án:** ResourcesManagement
- **Tên tiếng Việt:** Hệ thống quản lý thiết bị / Quản lý tài nguyên
- **Mô tả:** Hệ thống quản lý mượn/trả thiết bị công nghệ
